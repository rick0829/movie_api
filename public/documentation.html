Documentation for myFlix

Description URL HTTP Method Request body data format Response body data format
Return a list of movies to the user /movies GET NONE A JSON object holding data about all ofthe movies.
Return data about a single movie by title to the user /movies/[movieTitle] GET NONE A JSON object holding data, containing title, genre, description, and director of a single movie by title to the user
Return data about a genre by name /movies/genre/[genreTitle] GET NONE A JSON object holding data about the movie's genre. 
            
              {
                name: '',
                description: '',
              }
Return data about a director /movies/directors/[directorsName] GET NONE A JSON object holding data about a director. 
              
                {
                  name: '',
                  bio: '',
                  birth: '',
                  death: '',
                }
Allow new users to register /users POST A JSON object holding data about a user to add, structured like following: 
            
              {
                username: '',
                password: '',
                email: '',
                birthday: '',
                favoriteMovies: []
              }
A JSON object holding data about a user that was added, including an ID: 
            
              {
                id: '',
                username: '',
                password: '',
                email: '',
                birthday: '',
                favoriteMovies: []
              }
Allow users to update their user info /users/[id] PUT NONE A message indicating their user info was sucessfully updated
Allow users to add a movie to list of favorites /movies/[id]/[movieTitle] POST NONE A message indicating that the movie was sucessfully added to the their list.
Allow users to remove a movie from list of favorites /movies/[id]/[movieTitle] DELETE NONE A message indicating the movie was sucessfully removed.
Allow exisiting users to deregister /users/[id] DELETE NONE A message indicating the user was sucessfully deregistered.
